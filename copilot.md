# Copilot Notes

This project includes a CLAUDE.md file, which provides detailed information about the project structure, commands, and guidelines. Refer to CLAUDE.md for comprehensive documentation and instructions.

## Key Points from CLAUDE.md
- **Project**: Personal blog by Karl Mayer, built with Eleventy (11ty) v3.
- **Commands**:
  - `pnpm dev`: Starts the development server with live reload.
  - `pnpm build`: Builds the project for production.
  - `pnpm lint`: Runs linting tools (ESLint and Stylelint).
  - `pnpm lint:fix`: Automatically fixes lint issues.
- **Structure**:
  - `pages/`: Contains layouts, CSS, and content files.
  - `dist/`: Build output (not committed to version control).
- **Writing Blog Posts**:
  - Create a new folder in `pages/blog/` with the format `YYYYMMDD-slug/`.
  - Add a `post.md` file with required front matter.
  - Include images in the same folder as the post.

For more details, consult the CLAUDE.md file directly.
